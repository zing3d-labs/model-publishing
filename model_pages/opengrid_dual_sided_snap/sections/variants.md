## Variants

All 16 combinations of Side A and Side B thickness and directionality are included. File names follow the pattern `a_[std|lite]_[dir|nondir]_b_[std|lite]_[dir|nondir].stl`.

| File | Side A | Side B |
|------|--------|--------|
| a_std_nondir_b_std_nondir | Standard, Nondirectional | Standard, Nondirectional |
| a_std_dir_b_std_nondir | Standard, Directional | Standard, Nondirectional |
| a_std_nondir_b_std_dir | Standard, Nondirectional | Standard, Directional |
| a_std_dir_b_std_dir | Standard, Directional | Standard, Directional |
| a_lite_nondir_b_std_nondir | Lite, Nondirectional | Standard, Nondirectional |
| a_lite_dir_b_std_nondir | Lite, Directional | Standard, Nondirectional |
| a_lite_nondir_b_std_dir | Lite, Nondirectional | Standard, Directional |
| a_lite_dir_b_std_dir | Lite, Directional | Standard, Directional |
| a_std_nondir_b_lite_nondir | Standard, Nondirectional | Lite, Nondirectional |
| a_std_dir_b_lite_nondir | Standard, Directional | Lite, Nondirectional |
| a_std_nondir_b_lite_dir | Standard, Nondirectional | Lite, Directional |
| a_std_dir_b_lite_dir | Standard, Directional | Lite, Directional |
| a_lite_nondir_b_lite_nondir | Lite, Nondirectional | Lite, Nondirectional |
| a_lite_dir_b_lite_nondir | Lite, Directional | Lite, Nondirectional |
| a_lite_nondir_b_lite_dir | Lite, Nondirectional | Lite, Directional |
| a_lite_dir_b_lite_dir | Lite, Directional | Lite, Directional |
