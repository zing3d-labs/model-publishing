## Print Settings

All parts print flat with no supports required.

- **Layer height:** 0.2mm
- **Walls/perimeters:** 2 is ok, more for load bearing parts if needed.
- **Infill:** 15% (gyroid or grid)
- **Material:** PLA works great; PETG for higher-load applications

The snap connectors require clean, accurate prints to function correctly. Avoid excessive speed on those parts — standard profile speeds work well.
